# Beta Calculator

## 概述

该Skill用于计算股票的滚动Beta系数，基于过去100个交易日的日回报率数据。Beta系数衡量个股相对于市场的系统性风险，是CAPM模型的核心参数。

## 功能特性

- **滚动窗口计算**: 基于过去100个交易日计算滚动Beta
- **市场基准**: 使用沪深300指数作为市场基准
- **批量处理**: 自动处理文件夹中所有股票数据文件
- **自动保存**: 直接在原文件中添加Beta列
- **统计信息**: 提供Beta的均值、标准差等统计信息

## Beta系数公式

```
Beta = Cov(Ri, Rm) / Var(Rm)

其中：
- Ri: 个股的日回报率（简单回报率）
- Rm: 市场（沪深300）的日回报率
- Cov(Ri, Rm): 个股与市场的协方差
- Var(Rm): 市场的方差
```

## 使用方法

### 基本用法

```bash
python3 calculate_beta.py
```

### 指定数据文件夹

脚本默认处理"靶机数据"文件夹中的股票数据。如需修改，请编辑脚本中的`data_dir`变量。

### 数据要求

输入数据应包含以下列：
- **列114**: 个股日回报率（Simple_Return）
- **列38**: 沪深300涨跌幅（PctChange，单位为%）

数据文件应包含3行表头，从第4行开始为数据行。

## Beta系数解读

| Beta值范围 | 含义 | 投资特性 |
|-----------|------|---------|
| Beta > 1.5 | 高波动股 | 进攻型，波动大于市场1.5倍以上 |
| Beta = 1.0 ~ 1.5 | 中高波动股 | 与市场波动同步或略高 |
| Beta = 1.0 | 市场同步股 | 波动与市场完全同步 |
| Beta = 0.5 ~ 1.0 | 中低波动股 | 波动小于市场 |
| Beta < 0.5 | 低波动股 | 防御型，波动仅为市场一半以下 |
| Beta < 0 | 负相关股 | 与市场走势负相关 |

## 数学原理

### CAPM模型
Beta系数源于资本资产定价模型（CAPM）：

```
E(Ri) = Rf + Beta × (E(Rm) - Rf)

其中：
- E(Ri): 个股期望收益率
- Rf: 无风险收益率
- E(Rm): 市场期望收益率
- E(Rm) - Rf: 市场风险溢价
```

### Beta的经济含义
- Beta衡量系统性风险（市场风险）
- 无法通过分散化消除
- 反映个股对市场波动的敏感程度

## 注意事项

1. **窗口要求**: 至少需要30个有效数据点才能计算Beta
2. **数据质量**: 脚本会自动过滤NaN值和无效数据
3. **文件覆盖**: 脚本会直接修改原文件
4. **市场数据**: 确保沪深300涨跌幅数据完整
